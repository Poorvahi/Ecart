E-Cart is a full-stack electronics e-commerce web application developed using Java Servlets, JSP, and JDBC, designed to deliver a seamless and interactive online shopping experience. 
The platform enables users to explore and purchase electronic products such as mobiles, laptops, and accessories through a user-friendly interface.
The application supports secure user authentication, dynamic product browsing, category-based filtering, and real-time cart management. 
It also includes order processing and email-based OTP verification, ensuring both functionality and security in user transactions.
On the administrative side, the system provides comprehensive management features, allowing admins to add, update, and delete products, categories, and users, as well as monitor orders and customer activity.
The backend is powered by JDBC for efficient database connectivity and data handling.

🚀 **Key Features-**
🔐 User Registration & Login with Authentication
📱 Product Browsing (Mobiles, Laptops, Accessories)
🔎 Search Functionality for Products
🗂️ Category & Subcategory Filtering
🛒 Add to Cart & Cart Management
📦 Order Placement & Tracking
📧 Email & OTP Verification System
💳 Secure Payment Integration
🛠️ Admin Dashboard for Product & User Management

🚀 **Tech Stack**
Frontend: HTML, CSS, JavaScript, JSP
Backend: Java Servlets
Database: JDBC (MySQL/Oracle)
Server: Apache Tomcat

🎯 Objective
To develop a robust and scalable e-commerce platform that demonstrates real-world implementation of web technologies, backend logic, and database integration using Java-based frameworks.
